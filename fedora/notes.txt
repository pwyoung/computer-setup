NVIDIA:
  - Docs
    - https://www.if-not-true-then-false.com/2015/fedora-nvidia-guide/
  - Get driver
    - https://www.nvidia.com/Download/index.aspx
  - Deps
    - sudo dnf install kernel-devel kernel-headers gcc make dkms acpid libglvnd-glx libglvnd-opengl libglvnd-devel pkgconfig
  - Disable Nouveau
    - sudo su -
      echo 'blacklist nouveau
blacklist lbm-nouveau
options nouveau modeset=0
alias nouveau off
alias lbm-nouveau off
' > /etc/modprobe.d/nouveau-blacklist.conf
  - Update grub2
    - /boot/efi/EFI/fedora/grub.cfg
  - Remove xorg-x11-drv-nouveau
    - dnf remove xorg-x11-drv-nouveau
  - Generate initramfs
    - mv /boot/initramfs-$(uname -r).img /boot/initramfs-$(uname -r)-nouveau.img
    - dracut /boot/initramfs-$(uname -r).img $(uname -r)
  - Setup runlevel 3
    - systemctl set-default multi-user.target  
    - reboot
  - Continue
    - dnf install lynx
      lynx https://www.if-not-true-then-false.com/2015/fedora-nvidia-guide/
  - Run NVIDIA Binary
    - chmod +x NVIDIA-Linux-x86_64-430.50.run
      ./NVIDIA-Linux-x86_64-430.50.run --help
  - Reset GUI mode
    - systemctl set-default graphical.target
      reboot
  -  2.10 VDPAU/VAAPI support
     - dnf install vdpauinfo libva-vdpau-driver libva-utils
